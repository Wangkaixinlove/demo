# MarkDown相关问题整理  
1.MarkDown是什么？    
是一个Web上使用的文本到HTML的转换工具，可以通过简单、易读易写的文本格式生成结构化的HTML文档。  
2.MarkDown的特点？      
+ 纯文本内容，兼容所有的文本编辑器与文字处理软件  
+ 可以放到版本管理系统中，追踪历史变更  
+ 轻松的导出HTML、PDF和本身.md文件  
+ 简洁 高效 可读 直观 学习成本低  
+ 排版样式简单  
3.MarkDown的用途？    
IT人士：写日志、技术文稿、记录代码片段、撰写文档  
科研人员/学生：撰写科技论文、记笔记  
4.MarkDown的编辑工具有哪些？  
MarkdownPad 有道云笔记 github等  
5.MarkDown的区块元素和区段元素分别包含哪些？  
 区块元素：  
1.列表  
有序列表 数字+.  
无序列表 减号/加号+空格    
2.段落与换行
换行 两个空格    
段落 前后各加一个空格   
3.标题  
 一级标题 #号后面加空格  
 二级标题  ##
4.代码区块
tab键或者四个空格
5.分层
>一层缩进
>>两层缩进 
 区段元素：   
1.强调   
**字体加粗** 左右各两个星号  
*斜体* 左右各一个星号  
2.链接  
<网址链接>  
[百度首页](http://www.baidu.com)文字链接  
3.代码
``高亮显示 键盘第二行第一个键  
4.图片
！[替代文本](图片地址)添加图片
